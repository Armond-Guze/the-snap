# Power Rankings Rewrite Format

Use this for power rankings rewrites only.

## Output order

Return results in this exact order:

1. `Title:`
2. `Homepage Display Title:`
3. `Slug:`
4. `Summary:`
5. `Meta Description:`
6. `Topic Hubs:`
7. `Tag References:`
8. `Teams:`
9. `Biggest Riser:`
10. `Biggest Faller:`
11. `Top-Level Intro:`
12. `Top-Level Conclusion:`
13. `Body:`

## Default choices

- Default `Topic Hubs` to `NFL Power Rankings`
- Choose `1–2` good `Tag References`

## Rewrite rules

- Rewrite in original wording
- Do not mirror the source structure
- Keep all claims grounded in the source text
- If a fact is unclear, write `Needs verification`
- Remove source clutter like promos, ads, sidebars, and unrelated inserts
- If the source is incomplete, keep the available ranking material and mark missing context as `Needs verification`

## Required rules

- Always include `Top-Level Intro`
- Always include `Top-Level Conclusion`

## Slug rules

- lowercase only
- hyphen-separated
- usually `6–10` words
- avoid dates unless needed

## Field guidance

- `Summary` should be short and punchy
- `Meta Description` should be one accurate sentence
- `Biggest Riser` and `Biggest Faller` should be explicit team names when movement is known
